Holi Painter is a pastel-styled casual puzzle game where you control a cute character who throws color powders, mixes colors, and paints the world back to life.
Designed in a Korean casual-art style, the game is light, fun, and perfect for mobile.

Holi Painter는 파스텔 톤의 캐주얼 퍼즐 게임으로, 귀여운 캐릭터를 조작해 색 가루를 던지고 색을 섞어 세상을 다시 아름답게 색칠하는 게임입니다.
한국 감성의 캐주얼 아트 스타일로 디자인되어 가볍고 재미있으며 모바일에 딱 맞는 힐링 퍼즐 게임입니다.

If needed, the game can also run offline and includes simple endless-mode mechanics inspired by the Chrome Dino Run game — making it playable even with low network or limited hardware.
